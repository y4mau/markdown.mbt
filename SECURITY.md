# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through
[GitHub's private vulnerability reporting](https://github.com/y4mau/markdown.mbt/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

You should receive an initial response within 72 hours. We will work with you to
understand the issue and coordinate a fix and disclosure timeline.

## Supported Versions

Only the latest release on the `main` branch is supported with security updates.
